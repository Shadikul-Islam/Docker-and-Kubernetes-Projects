=========================
  Steps for Preparing
=========================


1. Create index lifecycle policy (Manually from Kibana)
2. Create Index Template (From Kibana Dev Tool)
3. Add Alias in Existing Index (From Kibana Dev Tool)


=================
  Special Notes
=================
1. During the Index Lifecycle Policy Maximum Age and Delete Age should be at least 2 minutes
for testing purposes.
2. The ILM Policy Name can be copied from logstash.conf file.
3. The ILM Policy Name will be same in logstash.conf file and Index Template file.

=====================
  Index Template
=====================
PUT _index_template/applog-index-template
{
  "template": {
    "settings": {
      "index": {
        "lifecycle": {
          "name": "applog-index-policy",
          "rollover_alias": "applog"
        }
      }
    }
  },
  "index_patterns": [
    "applog-*"
  ]
}


=================================
  Add Alias in Existing Index
=================================
POST /_aliases
{
  "actions": [
    {
      "add": {
        "index": "COPY-INDEX-NAME",
        "alias": "applog",
        "is_write_index": true
      }
    }
  ]
}
